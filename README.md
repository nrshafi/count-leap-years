
# Count Leap Year between two years

This Bash shell script identifies all leap years within a user-defined range and counts the total number of leap years.



## Features
- Takes starting year and ending year as input
- Displays all leap years in the given range
- Displays total leap year count



## Technologies Used
- Bash Shell Script



## ▶️ How to Run
```bash
chmod +x main.sh
./main.sh
